# 植物养护提醒功能说明

植物识别：豆包doubao-1.5-vision-pro-250328/在线推理
语音识别：百度

## 功能概述

植物养护提醒功能帮助用户及时进行植物养护，包括浇水、施肥、修剪等各项养护任务。系统会在设定的时间自动发送提醒消息，确保植物得到及时照顾。

## 主要功能

### 1. 提醒类型
- **浇水提醒**：定期提醒为植物浇水
- **施肥提醒**：定期提醒为植物施肥
- **光照调整**：提醒调整植物光照条件
- **修剪提醒**：定期提醒修剪植物
- **病虫防治**：提醒检查植物健康状况
- **换盆提醒**：提醒为植物换盆
- **自定义提醒**：用户可自定义养护事项

### 2. 提醒频率
- 每天
- 每2天
- 每3天
- 每周
- 每2周
- 每月
- 自定义

### 3. 通知方式
- **本地通知**：小程序内弹窗提醒
- **订阅消息**：微信订阅消息推送
- **声音提醒**：播放提示音
- **震动提醒**：设备震动提醒

## 技术架构

### 前端组件
- `utils/reminder.js`：提醒管理器
- `config/reminder.js`：提醒配置
- `pages/plantArchive/plantDetail.js`：植物详情页面

### 云函数
- `plantReminder`：处理提醒逻辑和发送订阅消息
- `plantReminderTrigger`：定时触发器

### 数据库集合
- `user_plants`：用户植物信息
- `plant_reminders`：植物养护提醒

## 安装和配置

### 1. 部署云函数
```bash
# 运行部署脚本
chmod +x deploy-reminder-functions.sh
./deploy-reminder-functions.sh
```

### 2. 配置订阅消息模板
1. 登录微信公众平台
2. 进入"功能 -> 订阅消息"
3. 申请植物养护提醒模板
4. 更新 `config/reminder.js` 中的模板ID

### 3. 配置云开发环境
确保 `app.js` 中的云开发环境ID正确配置。

## 使用说明

### 添加植物养护提醒
1. 进入植物详情页面
2. 点击"养护提醒"标签
3. 点击"添加养护提醒"
4. 选择养护类型
5. 设置提醒频率和时间
6. 输入提醒描述
7. 确认保存

### 管理提醒
- **启用/停用**：使用开关控制提醒状态
- **编辑**：修改提醒设置
- **删除**：删除不需要的提醒
- **查看即将到来的提醒**：在页面顶部查看最近的提醒

### 提醒通知
- 到达设定时间时，系统会自动发送提醒
- 用户会收到弹窗通知和Toast提示
- 支持声音和震动提醒
- 可发送微信订阅消息（需用户授权）

## 配置选项

### 提醒配置
在 `config/reminder.js` 中可以配置：
- 默认提醒时间
- 提醒提前时间
- 通知方式设置
- 频率选项

### 通知配置
```javascript
notification: {
  sound: '/audio/notification.mp3',  // 提示音文件
  vibration: true,                   // 是否震动
  showToast: true,                   // 是否显示Toast
  showModal: true                    // 是否显示弹窗
}
```

## 注意事项

1. **订阅消息权限**：首次使用需要用户授权订阅消息
2. **定时器限制**：小程序页面关闭后，本地定时器会失效，建议使用云函数定时检查
3. **存储限制**：本地存储有大小限制，注意及时清理过期数据
4. **云函数配额**：注意云函数的调用次数和资源使用情况

## 故障排除

### 提醒不工作
1. 检查云函数是否正常部署
2. 检查定时触发器是否配置正确
3. 检查数据库中的提醒数据是否正确
4. 检查用户是否授权了订阅消息

### 本地提醒失效
1. 确保小程序保持在前台运行
2. 检查本地存储是否正常
3. 重新设置提醒

### 订阅消息不发送
1. 检查模板ID是否正确
2. 检查用户是否授权
3. 检查云函数日志
